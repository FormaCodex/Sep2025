An Introduction of Paragraph Tag

In HTML, paragraphs are defined with the p tag.
A paragraph element in an HTML document is made up of content
such as text, images or other content
which appears between an opening p tag and closing p tag.

The p tag is the HTML tag for a paragraph. Adding the p tag to an HTML document,
followed by text, will create a paragraph on your page.
Adding the closing p tag will end the paragraph.

These breaks, that use "[br]" empty tags, break a line, like how in a word processor, hitting "enter" will break the line if it hasn't continued off the end and automatically broken. I will now demonstrate it with some Gundam Quotes and brs.
